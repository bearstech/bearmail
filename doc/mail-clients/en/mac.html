Configuration of Mail (Apple).



Informations about our mail account example:

E-Mail address: "mon.adresse@mondomaine.com"
Password: "a1b2c"
IMAP server: "mail.bearstech.com"
SMTP server: "mail.bearstech.com"


Configuration of Mail:

In the menu, click the "Mail" tab then click "Préférences".


A new window appears:

Click the "+" button on the down left corner of this window.


A new window appears:

"Type de compte :" set "IMAP".

"Description :" set "mon.adresse@mondomaine.com".

"Adresse électronique :" set "mon.adresse@mondomaine.com".

"Nom complet :" set "Mon nom".
You can set whatever you want in "Nom complet :"

"Serveur de réception :" set "mail.bearstech.com".

"Nom d'utilisateur :" set "mon.adresse@mondomaine.com".

"Mot de passe :" set "a1b2c".

"Serveur d'envoi (SMTP) :" set "Ajouter un serveur".


A new window appears:

"Serveur d'envoi" set "mail.bearstech.com".

Select "Utiliser SSL (Secure Sockets Layer)".

"Authentification :", set "Mot de passe".

"Nom d'utilisateur :", set "mon.adresse@mondomaine.com".

"Mot de passe :" set "a1b2c".

Click "OK".


Select the "Avancé" tab.

Select "Synchroniser automatiquement les BAL modifiées".

"Préfixe du chemin IMAP :" set "mail.bearstech.com".

Select "Utiliser SSL".

"Authentification :" select "Mot de passe".

Click on the red cross on the top left corner of the preferences window.


A new window appears:

Click "Enregistrer".



Mail is now ready for our e-mail account and connections will be crypted.
