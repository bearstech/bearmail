Configuration du client mail.



Configuration de thunderbird :

Pour télécharger Mozilla-Thunderbird, allez sur http://www.mozilla-europe.org/fr/products/thunderbird/


Une fois Thunderbird installé et exécuté, une fenêtre de configuration apparaît.
Si vous utilisez déjà Thunderbird, pour obtenir cette même fenêtre appruyez sur l'ongler "Fichier", puis "Nouveau", puis "Compte".

Selectionner si besoin "Ne rien importer", puis appuyer sur "Suivant".


Une nouvelle fenêtre apparaît :

Sélectionner "Compte courrier électronique", puis appuyer sur "Suivant".


Une nouvelle fenêtre apparaît :

Lorsque l'on compose un email, l'adresse d'envoie se fait sous la forme : "Nom <adresse@email>"
Ici la configuration de la partie "Votre nom :" correspond à la valeur de "Nom" pour l'adresse d'expéditeur "Nom <adresse@email>".
Dans notre exemple, le destinataire recevra les emails avec pour provenance "Mon nom <mon.adresse@mondomaine.com>", son contenu n'a pas d'importance mais Thunderbird a besoin d'une configuration pour pouvoir continuer.

La partie à définir pour "Adresse de courrier :" correspond à l'adresse email que l'on désire utiliser.
Dans notre exemple, l'adresse email à configurer est "mon.adresse@mondomaine.com".

Pour passer à l'étape suivante il faut cliquer sur le bouton "Suivant" situé en bas à droite.


Une nouvelle fenêtre apparaît :

Le serveur de réception utilisé est un serveur IMAP, il faut donc sélectionner "IMAP" et lui indiquer son adresse dans le champ "Nom du serveur :".
Dans notre exemple, le nom du serveur de réception est "imap.mondomaine.com".

Il faut aussi définir le nom du serveur d'envoi, pour se faire, remplir le champ "Nom du serveur :".
Dans notre exemple, le nom du serveur d'envoi est "smtp.mondomaine.com".

Pour passer à l'étape suivante il faut cliquer sur le bouton "Suivant" situé en bas à droite.


Une nouvelle fenêtre apparaît :

Pour pouvoir se connecter sur le serveur de réception, il faut s'identifier avec le nom de l'adresse email.
Pour se faire, il faut définir dans le champ "Nom d'utilisateur entrant :" le nom de l'adresse email authorisé.
Dans notre exemple, l'utilisateur entrant authorisé est "mon.adresse@mondomaine.com".

Pour pouvoir se connecter sur le serveur d'envoi, il faut aussi s'identifier avec le nom de l'adresse email.
Pour se faire, il faut définir dans le champ "Nom d'utilisateur sortant :" le nom de l'adresse email authorisé.
Dans notre exemple, l'utilisateur sortant authorisé est "mon.adresse@mondomaine.com".
Pour les utilisateurs utilisant déjà thunderbird, cette dernière partie n'est pas modifiable, nous reviendrons dessus plus loin.

Pour passer à l'étape suivante il faut cliquer sur le bouton "Suivant" situé en bas à droite.


Une nouvelle fenêtre apparaît :

Thunderbird permet de configurer plusieurs adresses emails afin de relever et d'envoyer des courriers électroniques.
Pour rendre l'utilisation plus aisée dans l'arboraissances des répertoires d'emails, il est conseillé d'utiliser le nom de l'adresse email pour chaque répertoires racines des configuration.
Pour se faire, il faut définir dans le champ "Nom du compte :" l'adresse email employé.
Dans notre exemple on nommera le répertoire racine de notre configuration "mon.adresse@mondomaine.com".

Pour passer à l'étape suivante il faut cliquer sur le bouton "Suivant" situé en bas à droite.


Une nouvelle fenêtre apparaît :

Il s'agit du récapitulatif de la configuration de l'adresse email.

Il faut maintenant affiner cette configuration afin de pouvoir la sécuriser.

Pour passer à l'étape suivante il faut cliquer sur le bouton "Terminer" situé en bas à droite.


Fenêtre principale de Thunderbird :


Pour configurer la connexion sécurisée aux serveurs, il faut modifier la configuration du compte actuel.
Pour se faire, cliquer sur l'onglet "Édition" situé en haut à gauche de la fenêtre principale de Thunderbird, puis dérouler le menu et aller dans "Paramètres des comptes...", ou pour les windaubiens: Outils : Paramètres des comptes..."

Une nouvelle fenêtre apparaît :


Pour configurer l'accès au serveur d'envoi, il faut aller dans "Serveur Sortant (SMTP)" situé tout en bas dans l'arborescence, à gauche de la fenêtre. Puis selectionner "mon.adresse@mondomaine.com - smtp.mondomaine.com" et appuyer sur le bouton "Modifier".

Une nouvelle fenêtre apparaît :

Selectionner soit "TLS" qui permet de se connecter sur le port classique 25, soit sur "SSL" qui permet de se connecter sur le port sécurisé 465, puis valider en appuyant sur le bouton "OK".


Il ne reste plus qu'à configurer l'accès au serveur de réception.
Aller dans "Paramètres serveur" se situant dans l'arborescence sous "mon.adresse@mondomaine.com", à gauche de la fenêtre.

Même cas que pour le serveur d'envoi, deux choix sont possible pour sécuriser la connexion.
Dans la partie "Paramètres de sécurité", soit sélectionner "TLS" pour utiliser le port classique "143" soit sélectionner "SSL" pour utiliser le port "993" exclusivement destinné aux connexions sécurisée.

Thunderbird est fin prêt et les connexions seront cryptées.




Outlook

Configuration d'Outlook express :

Pour configurer outlook express, exécutez le.
Si vous utilisiez déjà Outlook Express, cliquez sur l'onglet "Outils" puis sur "Comptes..." dans le menu de l'onglet.
Si vous n'avez jamais utilisé Outlook, nous vous recommandons vivement d'utiliser Thunderbird.


Une nouvelle fenêtre apparaît :

Cliquez sur le bouton "Ajouter" puis sur "Courrier..."

Une nouvelle fenêtre apparaît :

Lorsque l'on compose un email, l'adresse d'envoie se fait sous la forme : "Nom <adresse@email>"
Ici la configuration de la partie "Nom complet :" correspond à la valeur de "Nom" pour l'adresse d'expéditeur "Nom <adresse@email>".
Dans notre exemple, le destinataire recevra les emails avec pour provenance "Mon nom <mon.adresse@mondomaine.com>". On définira donc pour le champ "Nom complet :" la valeur "Mon nom".
Le contenu de ce champ n'a pas d'importance, mais Outlook a besoin d'une configuration pour pouvoir continuer.

Pour passer à l'étape suivante il suffit de cliquer sur le bouton "Suivant >" situé en bas à droite.


Une nouvelle fenêtre apparaît :

L'adresse email de notre exemple est "Mon nom <mon.adresse@mondomaine.com>".
Pour configurer Outlook avec cette adresse, il faut définir pour le champ "Adresse de messagerie" la valeur "mon.adresse@mondomaine.com".

Pour passer à l'étape suivante il suffit de cliquer sur le bouton "Suivant >" situé en bas à droite.


Une nouvelle fenêtre apparaît :

Il s'agit de la fenêtre de configuration pour le serveur de réception et d'envoi.

Le serveur de réception utilisé est un serveur IMAP, il faut donc sélectionner "IMAP" pour le champ "Mon serveur de messagerie pour courrier entrant est un serveur".

Dans notre exemple, le nom du serveur de réception est "imap.mondomaine.com".
On définira donc pour le champ "Serveur de messagerie pour courrier entrant (POP3,IMAP ou HTTP) : la valeur "imap.mondomaine.com".

Le nom du serveur d'envoi de notre exemple est "smtp.mondomaine.com, on définira donc pour le champ "Serveur de messagerie pour courrier sortant (SMTP) : la valeur "smtp.mondomaine.com".

Pour passer à l'étape suivante il suffit de cliquer sur le bouton "Suivant >" situé en bas à droite.


Une nouvelle fenêtre apparaît :

Cette fenêtre permet de définir l'indentifiant pour la connexion au serveur de réception et d'envoi.

L'identifiant et le nom du mail utilisé, soit "mon.adresse@mondomaine.com".
On définira donc pour le champ "Nom du compte :" la valeur "mon.adresse@mondomaine.com".
Puis on définira le mot de passe utilisé pour s'y connecté, celui fournit est "a1b2c".
On définira donc pour le champ "Mot de passe :" la valeur "a1b2c" qui sera masquée.
Puis on cochera l'option "Mémoriser le mot de passe" afin de ne pas avoir à le définir à chaque fois nécessaires.

Pour passer à l'étape suivante il suffit de cliquer sur le bouton "Suivant >" situé en bas à droite.


Une nouvelle fenêtre apparaît :

Outlook va chercher à mettre à jour l'arbre des dossiers du compte sur le serveur de réception.
Cliquez sur "Oui".

Actuellement Outlook est maintenant configuré pour se connecter sans crytage.
Pour remédier à celà, cliquez sur l'onglet "Outils" puis "Comptes...".


Une nouvelle fenêtre apparaît :

Sélectionnez "imap.mondomaine.com" puis appuyez sur le bouton "Propriétés".


Une nouvelle fenêtre apparaît :

Rennomer "imap.mondomaine.com" par "mon.adresse@mondomaine.com" afin que la racine de l'arbre puisse porter ce nom. Cela facilite la gestion de plusieurs adresses emails.

Allez dans l'onglet "Serveurs"

Cochez "Mon serveur requière une authentification"

Allez dans l'onglet "Avancé"

Cochez les 2 cases "Ce serveur nécessite une connexion sécurisée (SSL)" et mettez "465" pour "Courrier sortant (SMTP)".

Outlook est finalement configuré pour l'adresse email "Mon nom <mon.adresse@mondomaine.com>" et ce en connexion sécurisée.

Cliquez sur le bouton "OK" sité en bas de la fenêtre de dialogue.


Une nouvelle fenêtre apparaît :

Pour terminer, cliquez sur le bouton "Oui".
