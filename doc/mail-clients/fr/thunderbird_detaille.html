Configuration du client email Thunderbird.



Information de base de notre compte d'exemple :

Adresse email : "mon.adresse@mondomaine.com"
Mot de passe  : "a1b2c"
Serveur IMAP  : "mail.mondomaine.com"
Serveur SMTP  : "mail.mondomaine.com"


Configuration de Thunderbird :

Pour télécharger Mozilla-Thunderbird, vous pouvez aller sur http://www.mozilla-europe.org/fr/products/thunderbird/


Une fois Thunderbird installé et exécuté, une fenêtre de configuration apparaît.
Si vous utilisez déjà Thunderbird, pour obtenir cette même fenêtre appuyez sur l'onglet "Fichier", puis "Nouveau", puis "Compte".

Sélectionnez si besoin "Ne rien importer", puis appuyez sur "Suivant".


Une nouvelle fenêtre apparaît :

Sélectionnez "Compte courrier électronique", puis appuyez sur "Suivant".


Une nouvelle fenêtre apparaît :

Lorsque l'on compose un email, l'adresse d'envoie se fait sous la forme : "Nom <adresse@email>".
Ici la configuration de la partie "Votre nom :" correspond à la valeure de "Nom" pour l'adresse d'expéditeur "Nom <adresse@email>".
Dans notre exemple, le destinataire recevra les emails avec pour provenance "Mon nom <mon.adresse@mondomaine.com>", son contenu n'a pas d'importance mais Thunderbird a besoin d'une configuration pour pouvoir continuer.

La partie à définir pour "Adresse de courrier :" correspond à l'adresse email que l'on désire utiliser.
Dans notre exemple, l'adresse email à configurer est "mon.adresse@mondomaine.com".

Pour passer à l'étape suivante, cliquez sur le bouton "Suivant" situé en bas à droite.


Une nouvelle fenêtre apparaît :

Le serveur de réception utilisé est un serveur IMAP, il faut donc sélectionner "IMAP" et lui indiquer son adresse dans le champ "Nom du serveur :".
Dans notre exemple, le nom du serveur de réception est "mail.mondomaine.com".

Il faut aussi définir le nom du serveur d'envoi (SMTP), pour se faire, remplir le champ "Nom du serveur :".
Dans notre exemple, le nom du serveur d'envoi est "mail.mondomaine.com".

Pour passer à l'étape suivante, cliquez sur le bouton "Suivant" situé en bas à droite.


Une nouvelle fenêtre apparaît :

Pour pouvoir se connecter sur le serveur de réception, il faut s'identifier avec le nom de l'adresse email.
Pour se faire, il faut définir dans le champ "Nom d'utilisateur entrant :" le nom de l'adresse email authorisé.
Dans notre exemple, l'utilisateur entrant authorisé est "mon.adresse@mondomaine.com".

Pour pouvoir se connecter sur le serveur d'envoi, il faut aussi s'identifier avec le nom de l'adresse email.
Pour se faire, il faut définir dans le champ "Nom d'utilisateur sortant :" le nom de l'adresse email authorisé.
Dans notre exemple, l'utilisateur sortant authorisé est "mon.adresse@mondomaine.com".
Pour les personnes ayant déjà configuré thunderbird, cette dernière partie n'est pas modifiable, nous reviendrons dessus plus loin.

Pour passer à l'étape suivante, cliquez sur le bouton "Suivant" situé en bas à droite.


Une nouvelle fenêtre apparaît :

Thunderbird permet de configurer plusieurs adresses emails afin de relever et d'envoyer des courriers électroniques.
Pour rendre l'utilisation plus aisée dans l'arboraissances des répertoires d'emails, il est conseillé d'utiliser le nom de l'adresse email pour le répertoire racine de compte.
Pour se faire, il faut définir dans le champ "Nom du compte :" l'adresse email employé.
Dans notre exemple on nommera le répertoire racine de notre configuration "mon.adresse@mondomaine.com".

Pour passer à l'étape suivante, cliquez sur le bouton "Suivant" situé en bas à droite.


Une nouvelle fenêtre apparaît :

Il s'agit du récapitulatif de la configuration de l'adresse email.

Il faut maintenant affiner cette configuration afin de pouvoir la sécuriser.

Pour passer à l'étape suivante, cliquez sur le bouton "Terminer" situé en bas à droite.


Une nouvelle fenêtre apparaît :

Une fois la configuration de base terminée, Thunderbird va essayer de se connecter au serveur de réception afin de relever le courrier.
La connexion au serveur se fait à l'aide d'identifiants, composés du nom de l'adresse email (que l'on a défini tout à l'heure) et d'un mot de passe.
En essayant de relever le courrier, Thunderbird demande alors le mot de passe à définir. Dans le champ prévu à cet effet, définissez la valeure du mot de passe nécessaire. Dans notre exemple on mettra donc "a1b2c". Puis cochez la case "Utiliser le gestionnaire de mots de passe pour retenir ce mot de passe." afin de ne plus avoir cette demande de mot de passe.

Cliquez sur le bouton "OK" situé en bas à droite.


Une nouvelle fenêtre apparaît :

Il s'agit d'un rappel à propos de ma gestion des mots de passe.

Cliquez sur le bouton "OK" situé en bas à droite.


Fenêtre principale de Thunderbird :

Pour configurer la connexion sécurisée aux serveurs, il faut modifier la configuration du compte actuel.
Pour se faire, cliquez sur l'onglet "Édition" situé en haut à gauche de la fenêtre principale de Thunderbird, puis déroulez le menu et allez dans "Paramètres des comptes..." pour la version GNU/Linux. Pour les autres versions il s'agit de l'onglet "Outils" puis "Paramètres des comptes...".

Une nouvelle fenêtre apparaît :

Pour configurer l'accès au serveur d'envoi, il faut cliquer sur "Serveur Sortant (SMTP)" situé tout en bas dans l'arborescence, à gauche de la fenêtre. Puis selectionner "mon.adresse@mondomaine.com - smtp.mondomaine.com" et appuyer sur le bouton "Modifier".


Une nouvelle fenêtre apparaît :

Selectionnez soit "TLS" qui permet de se connecter sur le port classique 25, soit sur "SSL" qui permet de se connecter sur le port sécurisé 465, puis validez en appuyant sur le bouton "OK".


Il ne reste plus qu'à configurer l'accès au serveur de réception.
Allez dans "Paramètres serveur" se situant dans l'arborescence sous "mon.adresse@mondomaine.com", à gauche de la fenêtre.


Même cas que pour le serveur d'envoi, deux choix sont possibles pour sécuriser la connexion.
Dans la partie "Paramètres de sécurité", soit sélectionner "TLS" pour utiliser le port classique "143" soit sélectionner "SSL" pour utiliser le port "993" exclusivement destinné aux connexions sécurisée.

Pour en finir avec ces configurations basiques, cliquez sur le bouton "OK" situé en bas à droite.



Thunderbird est fin prêt, les connexions seront cryptées.
