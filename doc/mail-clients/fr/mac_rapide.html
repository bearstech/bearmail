Configuration du client email Mail (Apple).



Information de base de notre compte d'exemple :

Adresse email : "mon.adresse@mondomaine.com"
Mot de passe  : "a1b2c"
Serveur IMAP  : "mail.bearstech.com"
Serveur SMTP  : "mail.bearstech.com"


Configuration de Mail :

Dans le menu, cliquez sur l'onglet "Mail", puis sur "Préférences".


Une nouvelle fenêtre apparaît :

En bas à gauche de cette fenêtre, cliquez sur le bouton "+".


Une nouvelle fenêtre apparaît :

"Type de compte :" mettre "IMAP".

"Description :" mettre "mon.adresse@mondomaine.com".

"Adresse électronique :" mettre "mon.adresse@mondomaine.com".

"Nom complet :" mettre "Mon nom".
Le contenu du champ "Nom complet :" n'a pas d'importance.

"Serveur de réception :" mettre "mail.bearstech.com".

"Nom d'utilisateur :" mettre "mon.adresse@mondomaine.com".

"Mot de passe :" mettre "a1b2c".

"Serveur d'envoi (SMTP) :" mettre "Ajouter un serveur".


Une nouvelle fenêtre apparaît :

"Serveur d'envoi" mettre "mail.bearstech.com".

Cochez "Utiliser SSL (Secure Sockets Layer)".

"Authentification :", mettre "Mot de passe".

"Nom d'utilisateur :", mettre "mon.adresse@mondomaine.com".

"Mot de passe :" mettre "a1b2c".

Cliquez sur "OK".


Sélectionnez l'onglet "Avancé".

Cochez "Synchroniser automatiquement les BAL modifiées".

"Préfixe du chemin IMAP :" mettre "mail.bearstech.com".

Cochez "Utiliser SSL".

"Authentification :" sélectionnez "Mot de passe".

Fermez la fenêtre de préférence en cliquant sur le bouton rouge situé en haut à gauche.


Une nouvelle fenêtre apparaît :

Cliquez sur "Enregistrer".



Mail est fin prêt et les connexions seront chiffrées.
